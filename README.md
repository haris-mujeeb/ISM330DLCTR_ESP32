# ESP32 ISM330DLCTR Project

This project uses an ESP32 with an ISM330DLCTR sensor. This repository contains the firmware and hardware design files.

## Firmware

The firmware is located in the `firmware/platfromio` directory and is a PlatformIO project.

## Hardware

The KiCad design files are located in the `hardware/KiCad` directory. There are several versions of the board in this directory.
